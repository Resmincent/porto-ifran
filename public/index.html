About Me

Saya adalah mahasiswa semester akhir di Universitas Gunadarma, jurusan Informatika. Saya memiliki pengalaman magang sebagai Web Developer di PT Goorita dan sebagai IT Intern di PT Tokio Marine Life Insurance Indonesia.

Di Goorita, saya terlibat dalam pengembangan situs web perusahaan, sedangkan di Tokio Marine saya mendukung pengembangan aplikasi mobile berbasis Flutter. Kedua pengalaman ini meningkatkan keterampilan saya dalam teamwork, komunikasi profesional, serta penguasaan teknologi.

Saya menguasai beberapa bahasa pemrograman seperti PHP dan Dart, serta teknologi front-end seperti HTML, CSS, dan JavaScript. Saya juga terbiasa menggunakan framework seperti React, Typescript, Laravel, dan Flutter.

Di luar coding, saya suka mengikuti tren teknologi terbaru, bermain game dan suka berolahraga.
